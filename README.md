# 퀴즈앱

## 프로젝트 정보

### 시작하기 Getting Started

```
npm install
npm run start
```

- http://localhost:4000/ 로 접속

### 개발 환경 및 버전

1.  Node.js 18.12.1
2.  NPM 8.3.0
3.  react 18.2.0
4.  react-dom 18.2.0
5.  react-router-dom 6.3.0
6.  redux 4.2.0
7.  react-redux 8.0.2
8.  typescript 4.7.4
9.  styled-components 5.3.5
